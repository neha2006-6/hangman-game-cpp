# Hangman Game in C++

A fun console-based Hangman game built with C++!

## How to Play
- Guess the hidden word letter by letter
- You have 6 chances before the hangman is complete
- Try to guess the word before you run out of chances!

## How to Run
```bash
g++ hangman.cpp -o hangman
./hangman
```

## Features
- Random word selection
- Visual hangman drawing
- Tracks used letters
- Cybersecurity themed words
